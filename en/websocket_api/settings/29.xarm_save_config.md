# 29.xarm\_save\_config
 
Save Settings Interface.

Save settings to firmware, can request to save settings after setting modbus baud rate.



###  Request & Response

**Message**



```json
{
    "cmd": "xarm_save_config",
    "data": {},
    "id": "1"
}
```     
**Request**









**Response:**     



```
code=0->Success;
code!=0->Failed, refer to xarm_api_code;
```
