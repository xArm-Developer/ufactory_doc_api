### 2. Request & Response
**Request**

| **data** | **数据类型** | **必须** | **描述**                               | **示例值**        |
| -------- | ------------ |--------| ------------------------------------ | ----------------- |
| path     | string       | no     | Path to upload to the system | test/xarm6/config |
| file     | string       | no     | Binary data for uploading files         | -                 |

**Response** 

```
result:
    'ok' -> 成功
    'Invalid Args'无效的参数 -> 失败
success:
    1 -> 保存文件数据失败
    0 -> 上传成功
```
