# 34.xarm\_set\_mount\_direction

192.168.1.66:18333/ws

设置安装方向接口。\
设置安装方向。

### Message: <a href="#message" id="message"></a>

```
{
    "cmd": "xarm_set_mount_direction",
    "data": {
        'save': true, 
        'is_check': true,  
        'degrees': [90, 180]
    },
    "id": "1"
}
```

\


### Request： <a href="#request" id="request"></a>

**请求类型：Application/json**

* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号
