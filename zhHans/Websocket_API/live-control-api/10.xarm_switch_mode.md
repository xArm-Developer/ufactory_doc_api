# 10.xarm\_switch\_mode



设置模式：
设置Xarm模式。

### Message: 

```
#开启手动模式

{
    "cmd": "xarm_switch_mode",
    "data": {
        "mode": 2,
    },
    "id": "1"
}
```




### Request： 

* cmd
  * string，必填，请求接口
* date
  * Application/json，必填，请求数据
* id
  * string，必填，请求序列号


| data   | 数据类型 | 必须 | 描述                  | 示例值 |
|--------|----------|------|-----------------------|--------|
|mode | int   | 是   | 切换手动模式：2<br>       切换控制模式：0 | 2   |



### Respons:

```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```
